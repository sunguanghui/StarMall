# 星途补给站 · 功能清单

## 已实现功能

### 一、用户认证

- [x] 用户登录（JWT Token，24 小时有效）
- [x] 用户注册（用户名唯一校验，密码加密存储）
- [x] 修改个人密码（需验证旧密码，修改后自动登出）
- [x] 管理员重置任意用户密码
- [x] 自动登录（Token 本地持久化）
- [x] 退出登录
- [x] 前端路由守卫 + 后端接口双重权限验证

### 二、用户管理（管理员）

- [x] 用户列表（分页、按用户名/姓名搜索）

- [x] 编辑用户信息（姓名、邮箱、手机）
- [x] 重置用户密码
