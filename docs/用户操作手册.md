## 1. 系统概述

**星途补给站**是一套面向家庭的星际主题亲子积分奖励系统。管理员（舰长）通过发放「星辰币」来激励孩子，孩子可以用累积的「能量」在商城中兑换心仪奖品、提交心愿单，还可以通过完成任务大厅中的任务打卡赢取额外能量。

### 角色说明

| 角色 | 称号 | 权限范围 |
|------|------|---------|
| 超级管理员 | 👑 舰长 | 全部功能 |
| 普通管理员 | 领航员 | 发放星辰币、管理自己的任务、审批自己负责的打卡 |
| 普通用户 | 🚀 飞行员 | 浏览商城、兑换、打卡、心愿单 |

### 能量说明

| 名称 | 说明 |
|------|------|
| **总能量** | 历史累计获得的全部能量，只增不减 |
| **可用能量** | 当前可用于兑换商品的余额（总能量减去已消耗部分） |
| **已消耗能量** | 通过兑换商品花掉的能量 |

## 2. 账号管理

### 2.1 登录

访问系统首页，输入用户名和密码后点击「登录」。

> **初始账号（测试用）**
> - 超级管理员（舰长）：用户名 `admin`，密码 `admin123`
> - 普通用户：用户名 `zhangsan`，密码 `user123`

登录成功后自动跳转到首页仪表盘。

---

### 2.2 注册

在登录页点击「注册账号」进入注册页面。

| 字段 | 是否必填 | 要求 |
|------|----------|------|
| 用户名 | 必填 | 3～20 个字符，不可与已有账号重复 |
| 密码 | 必填 | 至少 6 位 |
| 确认密码 | 必填 | 需与密码完全一致 |
| 真实姓名 | 必填 | 用于显示和记录 |
| 邮箱 | 选填 | 格式需正确 |
| 手机号 | 选填 | — |

注册成功后自动跳转到登录页。

---

### 2.3 修改密码

点击页面右上角的头像，选择「修改密码」。

| 字段 | 要求 |
|------|------|
| 当前密码 | 必须填写正确的旧密码 |
| 新密码 | 至少 6 位 |
| 确认新密码 | 需与新密码一致 |

修改成功后系统会自动退出，需使用新密码重新登录。

---

### 2.4 头像设置

点击右上角头像 → 「个人中心」弹窗。

- **预设头像**：点击任意一款宇宙主题 emoji（🚀 👨‍🚀 🌟 🪐 🛸 ⭐）即可切换。
- **上传图片**：点击「上传本地图片」，支持 jpg/png/gif/webp，不超过 5MB。

## 3. 首页仪表盘

|------|----------|

## 4. 能量商城

点击左侧菜单「能量商城」进入。

### 浏览商品

页面以网格形式展示所有已上架商品。标有「🎁 盲盒」标签的商品为神秘盲盒，兑换后随机获得惊喜奖品。

### 兑换商品

1. 点击商品卡片下方的「立即兑换」按钮（盲盒显示「🎁 开盲盒」）。
2. 弹窗中确认可用能量、单件能量、库存和兑换数量。
3. 点击「确定」完成兑换。

**兑换限制说明：**

| 条件 | 结果 |
|------|------|
| 库存为 0 | 按钮显示「已售罄」，不可点击 |
| 可用能量不足 | 按钮置灰，不可点击 |
| 可用能量或库存只够兑换部分数量 | 兑换数量上限自动调整为最大可兑换件数 |

## 5. 我的能量

点击左侧菜单「我的能量」进入，查看完整的能量流水记录。

### 顶部统计

| 统计项 | 说明 |
|--------|------|
| 总能量 | 累计获得 |
| 可用能量 | 当前余额 |
| 已消耗 | 兑换消耗的能量 |
| 单星辰币数量 | 获得的单星辰币（+1能量）总次数 |
| 双星辰币数量 | 获得的双星辰币（+5能量）总次数 |

### 流水记录表

| 列名 | 说明 |
|------|------|
| 类型 | 单星辰币 / 双星辰币 / 扣除 |
| 能量 | 正数为获得，负数（红色）为扣除 |
| 原因 | 管理员备注的具体原因 |
| 舰长寄语 | 管理员附加的鼓励语（如有，显示为蓝色提示条） |
| 赋能官 ✨ | 操作的管理员名称（审计信息） |
| 时间 | 记录创建时间 |

> 扣除 5 分及以上的记录会显示 ☄️ 图标并以深红加粗样式标注；移动端卡片变为红色背景。

## 7. 星际心愿单

点击左侧菜单「星际心愿单」进入，向管理员申请想要的奖品。

### 提交心愿

点击「提交新心愿」：

| 字段 | 要求 |
|------|------|
| 心愿名称 | 必填，描述想要的奖品 |
| 期望能量 | 必填，至少为 1 |

### 心愿状态

| 状态 | 含义 |
|------|------|
| 待审核 | 已提交，等待管理员处理 |
| 已批准 | 管理员同意，已在商城创建对应商品（上架后即可兑换） |
| 已拒绝 | 管理员未通过本次申请 |

> 心愿被批准后，商品默认下架，舰长需手动将商品上架后才能在商城兑换。

## 8. 任务大厅

点击左侧菜单「任务大厅」进入，完成任务打卡赢取额外能量。

### 任务类型

| 类型 | 打卡限制 | 说明 |
|------|---------|------|
| 每日日常 | 每天限打卡1次 | 适合每天需要坚持的习惯 |
| 阶段里程碑 | 无次数限制 | 适合长期目标，每次完成均可提交 |

### 打卡流程

1. 切换到对应标签页（每日日常 / 阶段里程碑）。
2. 找到想完成的任务，点击「打卡」按钮。
3. 状态变为「审核中」，等待管理员审批。
4. 审批通过后自动发放能量，可在「我的记录」标签页和「我的能量」流水中查看。

### 打卡状态

| 状态 | 含义 |
|------|------|
| 审核中 | 打卡已提交，等待管理员审批 |
| 已完成 | 审批通过，能量已发放 |
| 已驳回 | 管理员驳回了本次打卡 |

## 9. 管理员功能

以管理员账号登录后，左侧菜单出现「管理中心」折叠菜单。

> **权限说明**：超级管理员（舰长）拥有全部管理权限；普通管理员（领航员）只能发放星辰币、管理自己创建的任务和审批自己负责的打卡。

---

### 9.1 用户管理（仅舰长）

路径：管理中心 → 用户管理

#### 新增用户

| 字段 | 要求 |
|------|------|
| 用户名 | 必填，不可重复 |
| 密码 | 必填 |
| 真实姓名 | 必填 |
| 邮箱 | 选填 |
| 手机号 | 选填 |
| 角色 | 普通用户 / 管理员 |
| 超级管理员 | 当角色为管理员时可选（仅舰长可设置） |
| 头像 | 选择预设或上传图片 |

#### 编辑用户

可修改真实姓名、邮箱、手机号、头像。用户名不可修改。密码留空则不修改。

#### 重置密码

点击「重置密码」，输入新密码后确认，立即生效。

---

### 9.2 发放星辰币

路径：管理中心 → 发放星辰币

#### 奖励/惩罚类型

| 类型 | 能量变化 | 说明 |
|------|----------|------|
| 单星辰币 ⭐ | +1 | 普通表扬 |
| 双星辰币 🚀 | +5 | 优秀表现 |
| 陨石撞击 | −2 | 轻度惩罚 |
| 星暴气流 | −5 | 中度惩罚 |
| 黑洞吞噬 | −10 | 重度惩罚 |
| 自定义扣除 | 自定义 | 手动输入扣除分值，与预设档位互斥 |

> 可用能量最低扣至 0，不会变为负数。

#### 操作步骤

1. 选择目标用户。
2. 选择星辰币类型（奖励）或扣除档位（惩罚），也可手动输入自定义分值。
3. 填写原因（必填）。
4. 可选填写「舰长寄语」，作为额外鼓励留给小朋友。
5. 点击「发放」完成操作。每笔记录自动关联「赋能官 ✨」审计信息。

---

### 9.3 商品管理（仅舰长）

路径：管理中心 → 商品管理

| 字段 | 要求 | 说明 |
|------|------|------|
| 商品名称 | 必填 | — |
| 商品描述 | 选填 | 显示在商品卡片上 |
| 商品图片 | 选填 | 支持 jpg/png/gif/webp，不超过 5 MB |
| 所需能量 | 必填，最小值 1 | 兑换所需能量数 |
| 库存数量 | 必填，最小值 0 | 为 0 时商城显示「已售罄」 |
| 是否盲盒 | 开关 | 开启后兑换时随机分配奖品 |
| 排序 | 最小值 0 | 数值越小排列越靠前 |
| 状态 | 上架 / 下架 | 下架商品不在商城展示 |

> 新建商品默认**下架**，需手动切换为上架后用户才可见。

---

### 9.4 兑换管理（仅舰长）

路径：管理中心 → 兑换管理

查看所有用户兑换记录。仅对「已完成」状态可执行取消操作，取消后能量全额退回、库存恢复。

---

### 9.5 心愿审核（仅舰长）

路径：管理中心 → 心愿审核

**批准**：心愿状态变「已批准」，自动在商品库创建对应商品（下架状态）。需前往商品管理手动上架。

**拒绝**：心愿状态变「已拒绝」，不创建任何商品。

---

### 9.6 任务管理

路径：管理中心 → 任务管理

#### 任务管理（上方卡片）

- 新增、编辑任务（名称、类型、奖励能量、是否启用）。
- 舰长可指定「专属审批人」；不指定则所有管理员均可审批。
- 普通管理员只能看到和管理自己创建的任务。

#### 打卡审批（下方卡片）

- 查看待审核打卡列表，支持按状态过滤。
- **批准**：自动发放能量，写入流水记录。
- **驳回**：带确认弹窗，驳回后不发放能量。
- 普通管理员只能看到自己负责审批的打卡（即任务的 `reviewer_id` 为自己或未指定审批人的任务）。

## 10. 常见问题

**Q：兑换按钮是灰色的，点不了？**  
A：可能是可用能量不足或商品库存为 0。请先查看「我的能量」页面确认当前可用能量。

**Q：兑换了盲盒，怎么知道抽到了什么？**  
A：前往「兑换记录」页面，找到对应记录，「备注」列会显示本次抽中的具体奖品名称。

**Q：心愿已批准，但商城里找不到对应商品？**  
A：商品批准后默认下架，需要管理员前往「商品管理」手动上架，上架后即可在商城看到。

**Q：我的可用能量被扣到 0 了，还会继续扣吗？**  
A：不会。系统设有保护机制，可用能量最低为 0，不会变为负数，但流水记录保留实际扣除值。

**Q：已完成的兑换可以退吗？**  
A：只有舰长可以在「兑换管理」页面取消已完成的兑换，取消后能量全额退回，库存也会恢复。

**Q：修改密码后为什么被强制退出了？**  
A：修改密码是安全操作，完成后系统会清除当前登录状态，需用新密码重新登录。

**Q：商品为什么不能删除？**  
A：若该商品存在任何兑换历史记录（无论状态），系统会拒绝删除以保护完整历史。可将其下架代替删除。

**Q：普通管理员（领航员）登录后看不到任务？**  
A：领航员只能看到自己创建的任务。需由舰长创建任务并将其指定为审批人，或领航员自行在任务管理中新建任务。

**Q：任务打卡后没有发放能量？**  
A：打卡提交后处于「审核中」状态，需等待管理员审批通过才会发放能量。请在「我的记录」标签页中查看审批进度。

**Q：每日日常任务今天已经打过卡了，还能再打吗？**  
A：不能。每日日常任务每人每天限打卡 1 次，系统会自动拦截重复提交。阶段里程碑任务无此限制。
