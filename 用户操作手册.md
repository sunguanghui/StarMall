## 1. 系统概述

**星途补给站**是一套面向小朋友的积分奖励与兑换平台。管理员通过发放「星辰币」来激励小朋友的表现，小朋友可以用累积的「能量」在商城中兑换心仪的奖品，也可以提交心愿单申请定制奖品。

### 能量说明

| 名称 | 说明 |
|------|------|
| **总能量** | 历史累计获得的全部能量，只增不减 |
| **可用能量** | 当前可用于兑换商品的余额（总能量减去已消耗部分） |
| **已消耗能量** | 通过兑换商品花掉的能量 |

## 2. 账号管理

### 2.1 登录

访问系统首页，输入用户名和密码后点击「登录」。

> **初始账号（测试用）**
> - 管理员：用户名 `admin`，密码 `admin123`
> - 普通用户：用户名 `zhangsan`，密码 `user123`

登录成功后自动跳转到首页仪表盘。

---

### 2.2 注册

在登录页点击「注册账号」进入注册页面。

| 字段 | 是否必填 | 要求 |
|------|----------|------|
| 用户名 | 必填 | 3～20 个字符，不可与已有账号重复 |
| 密码 | 必填 | 至少 6 位 |
| 确认密码 | 必填 | 需与密码完全一致 |
| 真实姓名 | 必填 | 用于显示和记录 |
| 邮箱 | 选填 | 格式需正确 |
| 手机号 | 选填 | — |

注册成功后自动跳转到登录页。

---

### 2.3 修改密码

点击页面右上角的头像，选择「修改密码」。

| 字段 | 要求 |
|------|------|
| 当前密码 | 必须填写正确的旧密码 |
| 新密码 | 至少 6 位 |
| 确认新密码 | 需与新密码一致 |

修改成功后系统会自动退出，需使用新密码重新登录。

## 3. 首页仪表盘

登录后默认进入首页仪表盘，展示以下内容：

### 彩虹统计卡片（4 张）

| 卡片 | 数据来源 |
|------|----------|
| 总能量 | 历史累计获得的全部能量 |
| 可用能量 | 当前可用于兑换的余额 |
| 获得星辰币 | 收到的星辰币记录总条数 |
| 兑换次数 | 已完成兑换的总次数 |

### 最近获得的星辰币

显示最新 5 条星辰币记录，包含类型、能量值、原因和时间。能量值为负数（红牌）时以红色标注。点击「查看更多」跳转至「我的能量」页面。

### 最近兑换记录

显示最新 5 条兑换记录，包含商品名称、消耗能量、状态和时间。点击「查看更多」跳转至「兑换记录」页面。

### 能量收集趋势图

折线图展示近期累计能量的增长趋势。

## 4. 能量商城

点击左侧菜单「能量商城」进入。

### 浏览商品

页面以网格形式展示所有已上架商品，每张商品卡片显示：

- 商品图片（无图时显示默认占位）
- 商品名称
- 商品描述
- 所需能量（⚡ 符号标注）
- 当前库存

标有「🎁 盲盒」标签的商品为神秘盲盒，兑换后随机获得惊喜奖品。

### 搜索商品

在顶部搜索框输入关键词后按回车，或点击清空按钮恢复全部商品。

### 兑换商品

1. 点击商品卡片下方的「立即兑换」按钮（盲盒显示「🎁 开盲盒」）。
2. 弹窗中确认以下信息：
   - 当前可用能量
   - 单件所需能量
   - 当前库存
   - 兑换数量（可调整，最大值受库存和可用能量双重限制）
3. 点击「确定」完成兑换。

**兑换限制说明：**

| 条件 | 结果 |
|------|------|
| 库存为 0 | 按钮显示「已售罄」，不可点击 |
| 可用能量不足 | 按钮置灰，不可点击 |
| 可用能量或库存只够兑换部分数量 | 兑换数量上限自动调整为最大可兑换件数 |

兑换成功后，可用能量立即扣减，页面自动刷新。

## 5. 我的能量

点击左侧菜单「我的能量」进入，查看完整的能量流水记录。

### 顶部统计

| 统计项 | 说明 |
|--------|------|
| 总能量 | 累计获得 |
| 可用能量 | 当前余额 |
| 已消耗 | 兑换消耗的能量 |
| 单星辰币数量 | 获得的单星辰币（+1能量）总次数 |
| 双星辰币数量 | 获得的双星辰币（+5能量）总次数 |

### 流水记录表

| 列名 | 说明 |
|------|------|
| 类型 | 单星辰币 / 双星辰币 / 红牌警告 |
| 能量 | 正数为获得，负数（红色）为扣除 |
| 原因 | 管理员备注的具体原因 |
| 舰长寄语 | 管理员附加的鼓励语（如有） |
| 发放人 | 操作的管理员名称 |
| 时间 | 记录创建时间 |

## 7. 星际心愿单

点击左侧菜单「星际心愿单」进入，向管理员申请想要的奖品。

### 提交心愿

点击「提交新心愿」：

| 字段 | 要求 |
|------|------|
| 心愿名称 | 必填，描述想要的奖品 |
| 期望能量 | 必填，输入愿意花费的能量数值（至少为 1） |

提交后心愿状态为「待审核」。

### 心愿状态

| 状态 | 含义 |
|------|------|
| 待审核 | 已提交，等待管理员处理 |
| 已批准 | 管理员同意，并已在商城创建对应商品（上架后即可兑换） |
| 已拒绝 | 管理员未通过本次申请 |

> 心愿被批准后，管理员还需手动将商品上架，商品出现在商城后方可兑换。

## 8. 管理员功能

以管理员账号登录后，左侧菜单出现「管理中心」折叠菜单，包含以下子模块。

---

### 8.1 用户管理

路径：管理中心 → 用户管理

#### 查看用户列表

展示所有账号，字段包括用户名、真实姓名、邮箱、手机号、角色、总能量、可用能量、注册时间。支持顶部搜索框按用户名或姓名筛选。

#### 新增用户

点击「新增用户」，填写以下信息：

| 字段 | 要求 |
|------|------|
| 用户名 | 必填，不可重复 |
| 密码 | 必填 |
| 真实姓名 | 必填 |
| 邮箱 | 选填 |
| 手机号 | 选填 |
| 角色 | 普通用户 / 管理员 |

#### 编辑用户

点击操作列「编辑」按钮，可修改真实姓名、邮箱、手机号、角色。用户名创建后不可修改。密码留空则不修改。

#### 重置密码

点击操作列「重置密码」按钮，输入新密码后确认，立即生效。

---

### 8.2 发放星辰币

路径：管理中心 → 发放星辰币

此页面用于向小朋友的账号发放奖励或扣除能量。

#### 奖励类型

| 类型 | 能量变化 | 说明 |
|------|----------|------|
| 单星辰币 | +1 | 普通表扬 |
| 双星辰币 | +5 | 优秀表现 |
| 红牌警告 | −1 | 扣除惩罚（可用能量最低为 0，不会变为负数） |

#### 操作步骤

1. 在左侧表单中选择目标用户。
2. 默认为发放模式；若需扣除能量，开启「扣除能量」开关，类型自动切换为红牌警告。
3. 选择星辰币类型（发放模式下可选单/双）。
4. 填写原因（必填）。
5. 可选填写「舰长寄语」，作为额外鼓励附加给小朋友。
6. 点击「发放」完成操作。

右侧实时显示最新发放记录（最近 20 条），包含用户、类型、能量、原因、寄语、操作人和时间。

---

### 8.3 商品管理

路径：管理中心 → 商品管理

#### 商品列表

展示所有商品，包含图片、名称、所需能量、库存、盲盒标记、状态、排序值。

#### 新增 / 编辑商品

| 字段 | 要求 | 说明 |
|------|------|------|
| 商品名称 | 必填 | — |
| 商品描述 | 选填 | 显示在商品卡片上 |
| 商品图片 | 选填 | 支持 jpg/png/gif/webp，不超过 5 MB |
| 所需能量 | 必填，最小值 1 | 兑换所需能量数 |
| 库存数量 | 必填，最小值 0 | 为 0 时商城显示「已售罄」 |
| 是否盲盒 | 开关 | 开启后兑换时随机分配奖品 |
| 排序 | 最小值 0 | 数值越小排列越靠前 |
| 状态 | 上架 / 下架 | 下架商品不在商城展示 |

> 新建商品默认为**下架**状态，需手动切换为上架后用户才可见。

#### 上下架切换

在操作列点击「上架」或「下架」按钮，立即生效。

#### 删除商品

点击「删除」按钮。若该商品存在任何兑换记录，系统将拒绝删除以保护历史数据。

---

### 8.4 兑换管理

路径：管理中心 → 兑换管理

查看所有用户的兑换记录，字段包含用户名、商品、数量、消耗能量、状态、时间、备注。

#### 取消兑换

仅对状态为「已完成」的记录可操作。点击「取消」后：

- 兑换状态变更为「已取消」
- 消耗的能量**全额退回**至该用户的可用能量
- 对应商品**库存恢复**

---

### 8.5 心愿审核

路径：管理中心 → 心愿审核

查看所有用户提交的心愿，字段包含用户名、心愿名称、期望能量、状态、提交时间。

#### 批准心愿

点击「批准」按钮后，系统自动执行以下操作：

1. 心愿状态变更为「已批准」
2. 自动在商品库中创建一件对应商品：
   - 商品名称 = 心愿名称
   - 商品描述 = "来自 {用户真实姓名} 的星际心愿"
   - 所需能量 = 心愿中填写的期望能量
   - 库存 = 1
   - 状态 = **下架**

> **注意**：批准后商品默认为下架状态，需前往「商品管理」手动将其上架，小朋友才能在商城中看到并兑换。

#### 拒绝心愿

点击「拒绝」按钮，心愿状态变更为「已拒绝」，不创建任何商品。

## 9. 常见问题

**Q：兑换按钮是灰色的，点不了？**  
A：可能是可用能量不足或商品库存为 0，卡片上会有「已售罄」提示或按钮自动置灰。请先查看「我的能量」页面确认当前可用能量。

**Q：兑换了盲盒，怎么知道抽到了什么？**  
A：前往「兑换记录」页面，找到对应记录，「备注」列会显示本次抽中的具体奖品名称。

**Q：心愿已批准，但商城里找不到对应商品？**  
A：商品批准后默认为下架状态，需要管理员前往「商品管理」手动上架，上架后即可在商城看到。

**Q：我的可用能量被扣到 0 了，还会继续扣吗？**  
A：不会。系统设有保护机制，可用能量最低为 0，不会变为负数。

**Q：已完成的兑换可以退吗？**  
A：只有管理员可以在「兑换管理」页面取消已完成的兑换，取消后能量全额退回，库存也会恢复。

**Q：修改密码后为什么被强制退出了？**  
A：修改密码是安全操作，完成后系统会清除当前登录状态，需用新密码重新登录。

**Q：商品为什么不能删除？**  
A：若该商品存在任何兑换历史记录（无论状态），系统会拒绝删除，以保护完整的兑换历史。可将其下架代替删除。
