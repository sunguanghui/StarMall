# 👩‍🚀 宇航员形象与动效指南 (聚焦 frontend/src/views/Login.vue)

## 1. 纯 SVG 代码绘制 (Pure SVG Illustration)
- 请直接在 Vue 组件的 `<template>` 中使用内联 `<svg>` 标签绘制一个卡通宇航员。
- **风格要求**：Q版、圆润、可爱（Kawaii 风格）。色彩要明快，比如白色的胖乎乎宇航服，头盔玻璃面罩带有高光，衣服上可以有马卡龙粉或薄荷绿的线条点缀。
- **动作设计**：宇航员可以设计成抱着一颗星星，或者在向屏幕挥手。

## 2. 布局与定位 (Positioning)
- 将这个 SVG 宇航员通过绝对定位 (`position: absolute`) 放置在登录玻璃卡片的周围。
- **推荐位置**：可以设计成他正趴在登录卡片的上方边缘（探出头和手，像在偷看），或者在卡片的侧边处于失重悬浮状态。
- **响应式注意**：确保在移动端（小屏幕）下，宇航员会自动缩小或隐藏，绝对不能遮挡输入框和登录按钮。

## 3. 注入生命力 (CSS Floating Animation)
- 为这个 SVG 宇航员增加连续的微动效，增强失重感。
- 编写 CSS `@keyframes astronaut-float`，让宇航员实现：缓慢上下浮动（位移 `10px-15px`），并伴随极小幅度（`2deg` 左右）的左右摇摆。
