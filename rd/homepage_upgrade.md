# 🎨 首页酷炫童趣化重构指南 (聚焦 frontend/src 目录)

## 1. 布局重构：从平面卡片到星际地图
- 抛弃原本冷冰冰的直角网格布局。
- **改造要求**：将四个核心数据卡片（总能量、可用能量、总获得数、兑换次数）重新布局为一个**非线性的“星际地图”**。四个卡片不再是并排的，而是像宇宙中的四个**“能量星球”**，围绕一个中心点（例如“孩子名+驾驶舱”）散落分布。

## 2. 视觉升级：星球化与圆润 Soft UI
- 利用我们在全局调整过的圆角样式，将四个卡片改为极致圆润的形状，最好是接近圆形。
- **色彩替换**：将默认蓝色改为温暖明快的糖果色。例如：总能量用活力黄，可用能量用薄荷绿，总获得数用马卡龙粉，兑换次数用天空蓝。
- 请使用柔和的弥散阴影和背景模糊（`backdrop-filter: blur`），让星球卡片看起来软绵绵的。

## 3. 动态元素与背景：让宇宙动起来
- **静态背景改为动态星空**：在首页背景层增加一个基于 CSS（例如：使用两个背景图层配合 `background-position` 动画，或简单的 SVG 路径动画）实现的**慢慢旋转的动态星空背景**。
- **星球微动效**：为四个星球卡片增加轻微的、上下起伏的浮动动画（`animation: float 4s ease-in-out infinite`）。

## 4. 互动反馈：趣味悬停动画
- **悬停放大**：当鼠标悬停在某个“星球卡片”上时，该星球要微微放大（`transform: scale(1.08)`）。
- **点击仪式感**：点击卡片或点击“去兑换”按钮时，增加一个轻微的缩放或彩虹色闪烁反馈。
