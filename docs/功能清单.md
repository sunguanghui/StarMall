# 星途补给站 · 功能清单

## 已实现功能

### 一、用户认证

- [x] 用户登录（JWT Token，24 小时有效）
- [x] 用户注册（用户名唯一校验，密码加密存储）
- [x] 修改个人密码（需验证旧密码，修改后自动登出）
- [x] 管理员重置任意用户密码
- [x] 自动登录（Token 本地持久化）
- [x] 退出登录
- [x] 前端路由守卫 + 后端接口双重权限验证

---

### 二、用户管理（超级管理员）

- [x] 用户列表（分页、按用户名/姓名搜索）
- [x] 新增用户（指定角色、可设置是否超级管理员）
- [x] 编辑用户信息（姓名、邮箱、手机、头像）
- [x] 重置用户密码
- [x] 用户头像设置（预设 emoji 或上传本地图片）

---

### 三、多级管理员

- [x] 三级角色：超级管理员（舰长）/ 普通管理员（领航员）/ 普通用户
- [x] `is_super_admin` 字段控制超级管理员权限
- [x] 前端菜单按权限自动隔离（用户管理、商品管理、兑换管理、心愿审核仅舰长可见）
- [x] 后端 API 双重校验，越权返回 403

---

### 四、星辰币系统

#### 管理员操作
- [x] 发放单星辰币（+1 能量）⭐
- [x] 发放双星辰币（+5 能量）🚀
- [x] 阶梯惩罚：陨石撞击（-2）/ 星暴气流（-5）/ 黑洞吞噬（-10）
- [x] 自定义大额扣分（手动输入负整数，与预设互斥）
- [x] 舰长寄语（选填，显示在孩子流水记录中）
- [x] 赋能官审计：每笔记录关联操作管理员（`admin_id`）
- [x] 扣分保护：可用能量最低归零，不产生负债

#### 用户查看
- [x] 个人能量统计（总能量 / 可用能量 / 已消耗）
- [x] 单/双星辰币获得次数统计
- [x] 完整流水记录（含类型、能量变化、原因、舰长寄语、赋能官 ✨、时间）
- [x] 负数能量值红色高亮显示
- [x] 扣 5 分及以上：☄️ 图标 + 深红加粗，移动端卡片变红色背景
- [x] 舰长寄语以提示条形式展示

---

### 五、任务大厅

#### 任务模板管理（管理员）
- [x] 新增任务（名称、类型、奖励能量、是否启用）
- [x] 任务类型：每日日常（每天限打卡1次）/ 阶段里程碑（无限制）
- [x] 超级管理员可指定专属审批人（不指定则所有管理员均可审批）
- [x] 普通管理员只能管理自己创建的任务
- [x] 启用/停用任务切换

#### 打卡流程（用户）
- [x] 在任务大厅按类型标签页浏览任务
- [x] 提交打卡（每日任务当天重复提交被自动拦截）
- [x] 打卡状态：审核中 / 已完成 / 已驳回
- [x] 我的记录标签页查看历史打卡

#### 打卡审批（管理员）
- [x] 待审核队列（按状态过滤：待审核/已批准/已驳回）
- [x] 批准：自动发放能量并写入流水记录（含审计信息）
- [x] 驳回：带确认弹窗
- [x] 普通管理员只看到自己负责审批的打卡记录

---

### 十一、安全

- [x] 密码 scrypt 加密存储
- [x] JWT Token 认证，24 小时有效
- [x] 三级权限校验（前端路由守卫 + 后端 API 双重验证）
- [x] `requireAdmin` + `requireSuperAdmin` 路由守卫
- [x] SQLAlchemy ORM 防 SQL 注入
- [x] Vue 模板自动转义防 XSS
- [x] 文件上传扩展名白名单 + 大小限制
- [x] 扣分时可用能量底部归零保护

---

### 十二、部署

- [x] Docker Compose 一键部署（MySQL + Flask/Gunicorn + Nginx）
- [x] 所有服务配置 `TZ: Asia/Shanghai` 时区
- [x] MySQL 配置 `--default-time-zone='+08:00'`
- [x] 多阶段前端 Dockerfile（Node 构建 + Nginx 托管）
- [x] 环境变量集中管理（`.env`）
- [x] 数据库一次性初始化脚本（`database/init.sql`，含全量建表 + 测试数据）
- [x] 管理员密码重置脚本（`reset_admin.py`，容器内执行）
