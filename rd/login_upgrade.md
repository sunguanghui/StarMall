# 🚀 登录页视觉重构指南 (聚焦 frontend/src/views/Login.vue)

## 1. 打造“深度宇宙”动态背景
- **多层视差背景**：不要只用一个简单的渐变。请使用 CSS 的 `linear-gradient` 配合 `radial-gradient` 营造出带有星云感（Nebula）的深邃底色（紫色、深蓝、粉色交织）。
- **流星与繁星动效**：利用 CSS 伪元素 (`::before`, `::after`) 或多个背景图层，实现：
    - 随机闪烁的星星（星星大小不一，呼吸感闪烁）。
    - 划过屏幕的流星动画（每隔几秒随机位置划过一条光痕）。

## 2. 登录卡片的“玻璃拟态”改造 (Glassmorphism)
- **卡片风格**：登录框不要用纯白色死板卡片。请改为**半透明的玻璃质感**：
    - 使用 `backdrop-filter: blur(15px)` 实现背景模糊。
    - 边框采用极细的淡彩色半透明描边，模拟飞船窗户的质感。
    - 增加大圆角（建议 `30px` 以上）和柔和的弥散发光阴影。

## 3. 增加“浮动星际元素”
- **漂浮动效**：在登录框周围（或背景中）增加 1-2 个正在缓慢上下漂浮的元素。
    - 一个是之前提到的 `<Star />` (星星)。
    - 一个是 `<Ship />` (小火箭/飞船)。
- **动画要求**：使用 `keyframes` 编写一个 `float` 动画，让它们看起来像在失重空间中轻轻晃动。

## 4. 输入框与按钮的趣味化
- **图标点缀**：在用户名和密码输入框的 `prefix` 位置，确保使用了星际相关的图标。
- **登录按钮**：按钮要使用更醒目的色彩，并增加悬停时的“能量填充”感（例如轻微的缩放反馈和阴影扩散）。
