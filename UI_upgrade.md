# 角色与目标
你现在是 Claude Code。目前「星途补给站 (原大拇哥商城)」的后端逻辑和进阶功能（如扣分、盲盒等）已经全部开发完毕，运行完美。
现在的唯一核心任务是：**全局 UI 视觉与交互重构**。

目前前端基于 Element Plus 默认样式，视觉上过于像“企业级后台管理系统”，非常古板。由于最终用户是一个刚上一年级（约 6-7 岁）的小朋友，你需要通过修改 CSS 和组件属性，将其彻底改造成一个**充满童趣、色彩明快、圆润可爱、像小游戏一样**的儿童友好型界面。

# 🎨 UI/UX 改造指南 (请重点针对前端 frontend/src 进行修改)

## 1. 色彩体系 (Candy / Pastel Palette)
- 彻底抛弃默认的企业商务蓝。
- 引入明快、温暖的马卡龙色系/糖果色作为主色调（例如：活泼的暖黄、马卡龙粉、薄荷绿、天空蓝）。
- 请通过覆盖全局 CSS 变量（例如 `:root` 中的 `--el-color-primary` 等）或直接编写全局样式来实现色彩替换。

## 2. 极致圆润的形状 (Rounded Shapes & Soft UI)
- 将所有的卡片 (el-card)、按钮 (el-button)、输入框 (el-input)、标签 (el-tag) 和弹窗 (el-dialog) 的边角改为**大圆角**（例如设置 `border-radius: 16px` 或 `24px`，甚至按钮采用完全的药丸形状）。
- 去除生硬的边框线 (`border: none`)，改用**柔和的弥散阴影 (Soft Box Shadows)**，让所有卡片看起来像悬浮在空中的云朵或糖果。

## 3. 字体与排版 (Typography)
- 适当增大核心界面的字体大小（`font-size`），特别是数字面板和商品标题，方便小朋友阅读。
- 增加整体的留白（Padding / Margin），不要让界面显得像密密麻麻的报表。

## 4. 趣味悬停动效 (Playful Hover Animations)
- 为所有的商品卡片、可点击的按钮增加交互反馈动效。
- 例如：鼠标悬停时，卡片微微放大 (`transform: scale(1.05)`)，并伴随平滑的过渡 (`transition: all 0.3s ease-in-out`)，或者产生轻轻的上下浮动效果。

## 5. 重点改造页面指出
- **用户仪表盘 (`Dashboard.vue`)**：顶部的 4 个统计数据卡片（总能量、可用能量等）请分别赋予 4 种不同的明快背景色，做成彩虹卡片。
- **积分商城 (`Mall.vue`)**：商品卡片要做得像橱窗里的玩具，突出图片；兑换按钮要做得像游戏里的购买按钮（大且圆润，最好带渐变色）。
